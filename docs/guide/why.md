- 见识短浅：真心不知道已经有轮子的存在，不然谁傻，花这么多时间干这个事？
- 方便自己：自己的脚，很臭，那也是自己的脚

## 兼容 uniapp 全平台
意味着可以编译为：h5、app、uniapp x、鸿蒙、微信小程序、支付宝小程序、字节小程序... 

## 字体图标
- 摇树优化：仅会打包使用到的ttf图标
- 任意拓展：一个项目，多个ttf字体图标库
- 支持在线：配置ttf文件的url地址即可

## ide 插件

在 hbuilderx、vscode 中提供了编辑器插件，功能：

1. 鼠标悬停查看示意
2. 代码提示
3. style 转 class
4. rpx 和 px 互换
